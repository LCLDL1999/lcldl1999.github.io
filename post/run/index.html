2020-06-04 coding Hugo HomeBrew Build-up

這個Build-up系列紀錄了我建立這個網站的大致過程。

在上一章，已經介紹了如何得到個人的範例網站。這一章介紹如何把它運行在個人電腦上。

因為我使用的是Mac，所以這部分以Mac上運行為主。

 

首先開啟終端機，然後在Homebrew中下複製下列command。

https://brew.sh/

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

這樣就會開始運行下載程序，中途會要求輸入password，輸入後會開始下載。

當看到Installation_Successful後就代表安裝成功。

 

之後進入Hugo的網站，往下拉到這個地方。

在終端機輸入以下command。這樣就可以下載安裝Hugo

$ brew install hugo

要檢查是否安裝成功，只要輸入下面command就可以確認了。

hugo version

之後把上一章準備好的範例檔案，選擇後點擊右鍵按服務，點擊新增位於檔案夾位置的終端機視窗，輸入

hugo server

可以看到上圖顯示了一個網頁的連結。把他複製後在Google上貼上就可以在電腦上運行範本網站。

這樣就完成了網站的運行。之後只要把網頁改成為自己想要的樣子就可以了。

下一章將介紹如何利用Github架構個人網站。
