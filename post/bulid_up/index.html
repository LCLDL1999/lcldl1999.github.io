Hugo Themes

2020-06-03 coding Hugo Github Build-up

這個Build-up系列紀錄了我建立這個網站的大致過程。

這是第一章，介紹如何在獲得自己的網站的範例。之後便可以通過修改範例來建造屬於自己的網站。

首先，是上hugo網站尋找喜歡的主題。

https://gohugo.io/

點進Themes頁面，選擇適合自己的主題。

 

假設我選擇的是Hugo Future Imperfect Slim這個主題，點進去後點Download。

 

之後會連結到Github的網站，進入後點Clone or download，再點Download ZIP就可以進行下載了。

 

把檔案解壓縮後找到叫exampleSite的檔案暫存在桌面，之後在這個檔案中新增一個叫themes的檔案。

之後在themes中把之前下載的檔案搬進去，把名字中的master刪除，就可以準備在電腦運行這個網站了。

下一章將會介紹如何運行網站在自己的電腦上。
